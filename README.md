# React Simple Form

This is a boilerplate for React form.

Use custom hook to manage the form state. Re-rendering only the changed part of the field.

## Directory structure

```bash
.
├── _app.tsx
├── components
│   ├── index.tsx
│   ├── select.tsx
│   └── text.tsx
├── hooks
│   ├── form.ts # Custom hook to manage the form
│   └── store.ts
└── main.tsx
```

# How to setup

```bash
$ yarn install
```

# How to run

```bash
$ yarn dev
```
